# Event Management Project
Completed project location with all commits is located in "develop" branch.

# Used Architectures
1. N-Layer

# Used Technologies
1. C# 6.0 (LTS)
2. EF Core
3. ASP.NET Core
4. AutoMapping
5. JWT Bearer Authentication
6. MS-SQL (T-SQL)

# Database Connection String
